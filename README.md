# AMTPI - Transport Public Iași (Concept Redesign)

⚠️ **DISCLAIMER:** Acest proiect a fost realizat strict în scop educațional, ca proiect pentru facultate. **NU este site-ul oficial** al Asociației Metropolitane de Transport Public Iași (AMTPI) și nu are nicio afiliere cu instituția reală. Datele prezentate (orare, tarife) pot să nu fie cele actualizate la zi. Pentru informații oficiale, vă rugăm să vizitați site-ul lor: [amtpi.ro](https://www.amtpi.ro/).

## 📝 Despre proiect
Acest repository conține un concept de redesign pentru platforma web AMTPI, concentrat pe îmbunătățirea experienței utilizatorului (UX/UI) și accesibilitate. 

Proiectul include:
* **Interfață modernă (UI):** Design curat și complet responsive (Mobile & Desktop).
* **Căutare Dinamică:** Dropdown-uri interdependente realizate în Vanilla JavaScript (selectarea comunei filtrează automat traseele disponibile).
* **Afișare Orare:** Modul interactiv pentru vizualizarea orarelor (Tur/Retur, Luni-Vineri/Weekend).
* **Funcția "Următoarea Plecare":** Un script care calculează ora curentă a utilizatorului și evidențiază automat următorul autobuz disponibil pe traseu.
* **Modal Tarife:** Pop-up interactiv pentru achiziția de bilete și abonamente cu redirecționare inteligentă către platforma de plată (24pay).

## 🛠️ Tehnologii utilizate
* HTML5
* CSS3 (fără framework-uri externe, Custom CSS)
* JavaScript (Vanilla DOM Manipulation)
* FontAwesome (pentru iconițe)
